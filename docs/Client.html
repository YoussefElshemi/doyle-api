(async) urban(word, discordopt, pageopt) → {Promise}

Source:
  • Client.js, line 111
Returns the definition of a word in the urban dictionary
Example
// getting an urban dictionary definition, needs to be async.
const doyle = require('doyle');
const api = new doyle.Client()

await api.urban('word', true) // for discord which returns the first page
await api.urban('word', true, 2) // for discord which returns the second page
await api.urban('word') // for array of objects
Parameters:
Name Type Attributes Default Description
word String The word to look up
discord Boolean <optional>
 false If this is true, it will construct an embed for you
page Number <optional>
 0 If there is a number in the third parameter, it will generate an embed using the third page
Returns:
Once the promise is awaited, you get an array of all definitions, or if the second parameter is true, you get a discord message embed
Type
Promise
